Roboto Regular is used for Body copy in 16px / 1rem. The recommended size for best readability should not be less than this.

Line-height: 24px

As a general rule if possible, keep all sizes scaling in sequences of 4s. 4/8/12/16/20/24/28/32/36/40/44.

APPLICATIONS

Basic Structural Examples

PHOTO-DRIVEN

Images should be used to build a story and create context for the messaging.

MAIN MESSAGING

Main messages and headlines are always the most prominent typographic element in our Roboto Slab typeface.

SUPPORT MESSAGING

Taglines and supportive information are either in the secondary header size or subhead application and body copy usage.

LOGO

ThinkCERCA logo should always be present.

TEXTURE USAGE

Use the Spark texture to help convey lightness and optimism in classrooms.

PATTERN USAGE

Applying subtle pattern work that does not overshadow main messaging is encouraged.

ICONOGRAPHY

Iconography should always be used as supportive graphics for all messaging.

HORIZONTAL APPLICATION

Using the same graphic guidelines when creating horizontal oriented materials.
